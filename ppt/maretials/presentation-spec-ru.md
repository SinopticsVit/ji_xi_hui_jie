# Спецификация презентации возможностей (PPTX)

Документ описывает структуру, визуальное оформление и правила размещения изображений для анонимной презентации на основе `capabilities-ru.md` и `catalog-ru.md`.

Целевой результат: презентация 16:9 на русском языке, примерно 22 слайда, без упоминаний поставщиков, заводов, сертификатных коллажей, подробных SKU-миниатюр и изображений с видимыми китайскими иероглифами.

## 1. Назначение и ограничения

Презентация должна показывать производственные возможности как нейтральный обзор направлений:

- авиационные системы запирания, быстросъёмные механизмы, кабина, обтекатели, аварийный выпуск шасси;
- наземное авиационное оборудование (GSE);
- крепёж высокого класса для авиации, космоса и железнодорожного транспорта;
- НИОКР, контроль качества и производственные процессы в обобщённом виде.

Что исключить из будущей презентации:

| Исключить | Причина | Как заменить |
|---|---|---|
| Названия поставщиков, заводов, брендов, адреса, сайты, QR-коды | Презентация должна быть анонимной | Нейтральные формулировки: «производственная база», «поставщик крепежа», «компетенция» |
| Описания заводов, история, награды, сертификатные коллажи | Не относятся к продуктовой презентации и могут раскрывать источник | Краткая матрица компетенций без сканов документов |
| Подробные картинки отдельных SKU из `catalog-items/` | Пользователь просил не включать детальные картинки каталога | Общие обзорные изображения `slide-10.png` и `slide-11.png` после crop |
| Изображения с видимыми китайскими иероглифами | Не подходят для финального русского PPTX | Crop видимой зоны продукции, overlay, либо текстовый слайд без изображения |
| Выставки, гранты, HR, вузовские партнёры, корпоративная стратегия | Не нужны в продуктовой презентации | Не включать |

Нейтральные стандарты и термины допустимы, если они не привязаны к конкретному поставщику: AS9100, ISO 9001, CNAS, HB, Lock Bolt, MES/ERP, CMM.

## 2. Design Tokens

Стиль: авиационно-промышленный, чистый, без логотипов и декоративного брендинга.

| Параметр | Значение |
|---|---|
| Формат | 16:9, 13.333 x 7.5 in |
| Основной фон | `#FFFFFF` |
| Вторичный фон | `#F5F7FA` |
| Основной тёмный | `#0B2A4A` |
| Акцент | `#0078A8` |
| Светлый акцент | `#E8F4F8` |
| Линии и рамки | `#CBD5E1` |
| Основной текст | `#1F2A44` |
| Вторичный текст | `#64748B` |
| Заголовки | Segoe UI Semibold, 28-32 pt |
| Основной текст | Segoe UI, 16-18 pt |
| Таблицы | Segoe UI, 12-14 pt |
| Footer | Номер слайда справа внизу, без названия компании |

Постоянная сетка:

| Зона | Размер |
|---|---|
| Внешнее поле | 0.45 in |
| Header bar | 0.50 in |
| Отступ после заголовка | 0.25 in |
| Footer | 0.30 in |
| Радиус визуальных карточек | 4-6 px |
| Рамка изображений | 1 pt, цвет `#CBD5E1` |

Не использовать:

- логотипы и символику поставщиков;
- красно-золотые корпоративные палитры исходных PDF;
- мелкий текст поверх изображений;
- скриншоты с видимыми сайтами, QR-кодами и подписями на китайском.

## 3. Каталог макетов

### 3.1 Title

Используется для обложки и финального смыслового слайда.

```text
+----------------------------------------------------------+
|                                                          |
|              ПРОИЗВОДСТВЕННЫЕ ВОЗМОЖНОСТИ               |
|        Авиационные системы запирания и крепёж            |
|                                                          |
|        [тонкая акцентная линия #0078A8]                  |
|                                                          |
|                                                1 / 22    |
+----------------------------------------------------------+
```

Параметры:

- фон `#FFFFFF` или `#F5F7FA`;
- заголовок 34-38 pt;
- подзаголовок 18-20 pt;
- без фотографий.

### 3.2 Section

Разделитель между крупными блоками A и B.

```text
+----------------------------------------------------------+
|                                                          |
|  A                                                       |
|  Системы запирания и авиационные механизмы               |
|                                                          |
|  Короткое пояснение: 1 строка                            |
|                                                          |
+----------------------------------------------------------+
```

Параметры:

- фон `#0B2A4A`;
- индекс раздела 72 pt, белый;
- заголовок 30-34 pt, белый;
- без footer, без изображений.

### 3.3 SplitTextImage

Базовый макет для слайда с тезисами и одним изображением.

```text
+----------------------------------------------------------+
| Header bar                                               |
| Заголовок слайда                                         |
| --- accent line ---------------------------------------- |
|                                                          |
| +-----------------------+  +---------------------------+ |
| | Текстовая зона 42%    |  | Изображение 52%           | |
| | - тезис 1             |  | crop / рамка / подпись    | |
| | - тезис 2             |  |                           | |
| | - тезис 3             |  |                           | |
| +-----------------------+  +---------------------------+ |
|                                                N / 22    |
+----------------------------------------------------------+
```

Параметры:

- текст слева: 0.65-6.00 in;
- изображение справа: 6.40-12.75 in;
- высота изображения до 4.80 in;
- подпись под изображением 10-11 pt, серый цвет.

### 3.4 ImageHero

Макет для продуктового изображения на большую часть слайда.

```text
+----------------------------------------------------------+
| Header bar                                               |
| Заголовок слайда                                         |
| --- accent line ---------------------------------------- |
|                                                          |
|       +------------------------------------------+       |
|       |                                          |       |
|       |        Основное изображение 85%          |       |
|       |        crop + border                     |       |
|       |                                          |       |
|       +------------------------------------------+       |
|       Короткая подпись, 1 строка                         |
|                                                N / 22    |
+----------------------------------------------------------+
```

Параметры:

- изображение по центру, ширина 10.8-11.2 in;
- верхняя координата 1.45-1.65 in;
- подпись не должна повторять текст с картинки.

### 3.5 DualImage

Макет для двух общих обзорных кадров вместо 82 SKU-миниатюр.

```text
+----------------------------------------------------------+
| Header bar                                               |
| Типовая линейка - обзор                                  |
| --- accent line ---------------------------------------- |
|                                                          |
| +-------------------------+ +--------------------------+ |
| | crop slide-10           | | crop slide-11            | |
| | Запирающие системы      | | Комплекты и GSE          | |
| +-------------------------+ +--------------------------+ |
|                                                          |
| Семейства: HB, TR-K, TR-C, TR-D, TR-S, TR-B, TR-H, TR-Q  |
|                                                N / 22    |
+----------------------------------------------------------+
```

Параметры:

- каждая картинка 47% ширины content zone;
- расстояние между картинками 0.35-0.45 in;
- изображения должны быть обрезаны до сетки продукции, без шапки исходного слайда.

### 3.6 Matrix

Макет для таблиц и сравнительных матриц.

```text
+----------------------------------------------------------+
| Header bar                                               |
| Заголовок таблицы                                        |
| --- accent line ---------------------------------------- |
|                                                          |
| +------------------------------------------------------+ |
| | Колонка 1 | Колонка 2 | Колонка 3                  | |
| |----------|----------|------------------------------| |
| | строка   | строка   | строка                       | |
| +------------------------------------------------------+ |
|                                                N / 22    |
+----------------------------------------------------------+
```

Параметры:

- header таблицы: `#0B2A4A`, белый текст;
- чётные строки: `#F8FAFC`;
- нечётные строки: `#FFFFFF`;
- без изображений.

## 8. Будущая сборка PPTX

Эта спецификация не создаёт `.pptx`. Если потребуется генерация, следующий шаг - отдельный скрипт `build_presentation_pptx.py` на `python-pptx`, который:

1. читает таблицу структуры из этого документа или содержит её как Python-структуру;
2. применяет crop к выбранным PNG;
3. вставляет изображения только из whitelist;
4. проверяет blacklist по путям;
5. сохраняет анонимную презентацию в формате `.pptx`.
