## 3. Каталог макетов

### 3.1 TitleBranded

Титульный слайд с логотипом, hero-фото и контактами. Используется для **слайда 1**.

```text
+----------------------------------------------------------+
| [logo]     ENGLISH COMPANY NAME                          |
|                                                          |
|              +------------------------+                  |
|              |   photo (Avito)        |                  |
|              |                        |                  |
|              +------------------------+                  |
|                                                          |
|  Address on Aviatsionnaya st.     email  |  dir. phone   |
+----------------------------------------------------------+
```

Параметры:

- логотип: верхний левый угол, ~0.8–1.0 in по высоте;
- название: по центру верхней трети, 30–36 pt, белый;
- фото: центр слайда, ширина ~60% content zone, рамка 1 pt `#CBD5E1`;
- footer: одна строка, 12–14 pt.

### 3.2 SplitIconsText

Три визуальных блока (круг/овал + прямоугольник) и текстовая колонка. Используется для **слайда 2**.

```text
+----------------------------------------------------------+
| Header bar                                               |
| Исследования, производство и международные поставки      |
| --- accent line ---------------------------------------- |
|                                                          |
|  [фото        [фото         |  - исследования и НИОКР     |
|   вертолёта]   самолёта]   |  - производство авиатехники |
|  [фото агро-               |  - международные поставки  |
|   самолёта]                |                             |
|                                                2 / 6     |
+----------------------------------------------------------+
```

Параметры:

- два круга сверху: ~1.2 in диаметр;
- овал/фрагмент з/ч снизу слева;
- текстовая зона справа: 45–50% ширины.

### 3.3 HeaderPlusTripleCards

Заголовок + подзаголовок + три карточки с таблицей, фото и инв. номером. Используется для **слайда 3**.

```text
+----------------------------------------------------------+
| Header bar                                               |
| Авиационные запчасти                          [блок A]   |
| --- accent line ---------------------------------------- |
| деятельность по поставке з/ч                             |
|                                                          |
| +----------------+ +----------------+ +----------------+ |
| | таблица з/ч    | | таблица з/ч    | | таблица з/ч    | |
| | + фото         | | + фото         | | + фото         | |
| | инв. №         | | инв. №         | | инв. №         | |
| +----------------+ +----------------+ +----------------+ |
| замки и запорные механизмы                    [блок B]   |
|                                                3 / 6     |
+----------------------------------------------------------+
```

Параметры:

- три равные колонки, gap 0.25 in;
- в каждой карточке: мини-таблица (2–4 строки), фото SKU, инвентарный номер 11–12 pt.

### 3.4 TripleMedia

Три медиа-блока в ряд с подписями. Используется для **слайда 4**.

```text
+----------------------------------------------------------+
| Header bar                                               |
| Фото и данные (TBD)                                      |
| --- accent line ---------------------------------------- |
|                                                          |
| +-------------+  +-------------+  +-------------+        |
| | аэродром    |  | з/ч из      |  | Excel /     |        |
| |             |  | презентации |  | таблица     |        |
| +-------------+  +-------------+  +-------------+        |
|   подпись          подпись            подпись            |
|                                                4 / 6     |
+----------------------------------------------------------+
```

Параметры:

- каждый блок ~30% ширины content zone;
- подпись под блоком: 10–12 pt, `#64748B`.

### 3.5 Title / SplitTextImage

Итоговый текстовый слайд, опционально с нейтральной иллюстрацией. Используется для **слайда 5**.
